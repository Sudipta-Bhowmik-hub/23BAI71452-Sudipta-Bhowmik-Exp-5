# 🎮 Smart Tic Tac Toe (Exp‑5)

## 📌 Introduction
Smart Tic Tac Toe is a React + Vite project built for **Experiment 5**.  
It demonstrates **Context API, Redux Toolkit, React hooks, and deployment on Vercel**.

---

## 🚀 Features
- Multiple pages: Home, Game, Analytics, About
- Context API for theme and difficulty
- Redux Toolkit for counter and analytics
- Tic Tac Toe game logic with winner calculation
- Difficulty selector (Easy/Medium/Hard)
- Theme toggle (Light/Dark)
- Deployed live on Vercel

---

## 🛠️ Technologies
- React + Vite  
- Redux Toolkit  
- Context API  
- CSS  
- Vercel
